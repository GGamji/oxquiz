변수란?

- 임의의 데이터를 저장하는 공간

변수 네이밍 규칙

  • 한글, 한자, 아랍어 등 모든 언어가 가능하지만, 영문자나 영단어만 사용
  • 대소문자는 구분해서 사용
  • 띄어쓰기 불가
  • 2개 이상의 단어를 조합하여 만드는 경우 '_'를 이용하거나 두번째단어부터 첫글자를 대문자로
  • '_'(언더바)를 제외한 특수 기호는 사용하지 않는다.
  • 숫자 사용 가능하지만, 숫자만 사용하거나 첫글자로 사용하지 않는다.
  • 자바스크립트 키워드, 예역어 사용금지
  • 사용목적을 알수 있는 단어를 사용

변수 키워드

변수를 사용하려면 선언부터 한다. 선언할 때 사용되는 키워드

var
- ES6이전 방식으로 사용하지 말 것
let
- 처음 선언할 때 한 번만 사용
- 값을 재할당할 때는 키워드는 생략
- 같은 이름의 변수는 사용할 수 없음
const
- 선언하면서 초기값 할당
- 값의 재할당 불가
- 값이 변경되는 데이터에서 사용할 수 없다.
